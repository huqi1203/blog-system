🔧 清除浏览器缓存指南

如果页面显示乱码，请按照以下步骤清除浏览器缓存

⚡ 快捷方法： 按 Ctrl + Shift + R (Windows) 或 Cmd + Shift + R (Mac) 强制刷新页面

Google Chrome

按 Ctrl + Shift + Delete (Windows) 或 Cmd + Shift + Delete (Mac)

选择时间范围："过去 1 小时" 或 "过去 24 小时"

勾选 "缓存的图片和文件"

点击 "清除数据"

Mozilla Firefox

按 Ctrl + Shift + Delete (Windows) 或 Cmd + Shift + Delete (Mac)

选择 "缓存"

点击 "立即清除"

🧭 Apple Safari

点击菜单栏 "开发" → "清空缓存"

如果没有"开发"菜单：设置 → 高级 → 勾选"在菜单栏中显示开发菜单"

或者按 Cmd + Option + E

Microsoft Edge

按 Ctrl + Shift + Delete

选择 "缓存的图片和文件"

点击 "立即清除"

📱 手机浏览器

Chrome (Android): 设置 → 隐私和安全 → 清除浏览数据

Safari (iOS): 设置 → Safari → 清除历史记录与网站数据

微信内置浏览器: 我 → 设置 → 通用 → 清空聊天记录 → 或直接卸载重装微信

✅ 清除后： 请重新访问 首页 或 管理后台

如有问题请联系管理员
