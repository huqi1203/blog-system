## 监控面板功能完善

### 修复内存监控数据显示问题
- 问题：中文 Linux 系统上 `free` 命令输出 "内存：" 而非 "Mem:"
- 解决：使用 `free -g | awk 'NR==2{print $2}'` 替代 `awk '/^Mem:/{...}'`
- 位置：/root/.openclaw/workspace/monitor-panel/server.py
- 现在可以正确显示内存使用数据

### 完善并发控制功能
- 新增 API 接口：

- 功能：支持手动调整并发数，实时显示槽位使用情况
