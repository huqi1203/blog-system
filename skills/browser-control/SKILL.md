# Browser Control Skill - 浏览器控制技能

## 描述
通过 OpenClaw 控制 Chrome 浏览器，实现自动化网页操作。

## 功能
- 打开任意网页
- 截图保存页面
- 点击元素
- 填写表单
- 滚动页面
- 获取页面内容

## 使用方法

### 基础命令

```
打开 https://example.com
截图
点击 "按钮文字"
输入 "用户名" 到 "输入框名称"
滚动到底部
```

### 快捷指令

| 指令 | 说明 |
|------|------|
| `打开 [网址]` | 打开指定网页 |
| `截图` | 截取当前页面 |
| `点击 [文字]` | 点击包含指定文字的元素 |
| `输入 [内容] 到 [位置]` | 在输入框中填写内容 |
| `搜索 [关键词]` | 在百度/Google 搜索 |
| `刷新` | 刷新当前页面 |
| `返回` | 返回上一页 |
| `前进` | 前进到下一页 |

## 配置要求

需要预先配置 Chrome Debug：
1. 运行 `/home/huqi/桌面/chrome-debug.sh` 启动浏览器
2. 确保调试端口 9222 可用
3. OpenClaw 配置中已添加 mydebug 浏览器配置

## 示例

### 示例 1：搜索信息
```
打开 https://www.baidu.com
输入 "OpenClaw 使用教程" 到搜索框
点击 "百度一下"
截图
```

### 示例 2：查看 GitHub 仓库
```
打开 https://github.com/huqi1203/blog-system
截图

```

### 示例 3：自动化操作
```
打开 https://example.com/login
输入 "username" 到用户名
```

## 注意事项

- 浏览器必须由用户手动启动（需要图形界面）
- 确保 Chrome Debug 在运行才能使用
- 部分网站可能有反爬虫机制
